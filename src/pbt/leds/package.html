Illustration de la notion de java bean au travers de la création d'une classe représentant une led.

Une led peut être allumée ou éteinte (propriété on) et peut changer de couleur (propriété color).

Une led est défini dans les classes *GLed* en fonction des particularités de développement. Le code est proposé en « version Swing » et en « version JavaFX » afin d'illustrer les différences entre les deux approches.
